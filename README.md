# dotfiles

My personal \*nix configuration files.

## What it includes

-   vim/nvim, tmux, tmate, sshrc, bash, zsh, alacritty, karabiner, kitty, mc, etc.
-   alias rewrites for various commands

## Installation

1. Clone to `$HOME`
2. Run `~/.dotfiles/install.sh`.
